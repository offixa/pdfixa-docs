# PDFixa Documentation

Docusaurus-based documentation site for [PDFixa](https://github.com/offixa/pdfixa) — deterministic PDF engine for Java.

## Run locally

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm run serve
```

Static output is in `build/`.
