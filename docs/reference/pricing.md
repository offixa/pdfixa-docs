PDFixa follows a **simple commercial licensing model**.

You can start with **PDFixa Core for free**, and upgrade to Pro when
your system requires advanced capabilities.

PDFixa Core remains completely open-source and available on Maven
Central.

You can generate production documents without any license.

Upgrade to **PDFixa Pro** when your application requires:

-   international language support
-   advanced layout
-   compliance features
-   enterprise support
